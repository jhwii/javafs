Mobile Game

넥슨의 자회사중 하나였다.
2005년 5월 넥슨은 모바일 게임 진출을 위해 모바일 게임 회사 엔텔리젼트를 인수하고 2006년 2월 사명을 넥슨모바일로 변경하였다.
넥슨 게임들의 IP를 이용한 모바일 게임들을 포함, 다양한 모바일 게임들을 출시하였다.
대표작으로는 게임빌의 게임빌 프로야구 시리즈, 컴투스의 미니게임천국 시리즈와 함께 2000년대에 누계 1000만 다운로드를 돌파한 모바일 게임 시리즈인 모바일 메이플스토리 시리즈가 있다.
이외에도 역전재판 시리즈나 태고의 달인 시리즈 같은 게임들을 피처폰에 이식시키기도 하였다.
